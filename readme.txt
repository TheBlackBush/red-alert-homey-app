Red Alert Israel brings real-time civil defense alert events into Homey so you can automate safety responses quickly. The app listens to public alert streams and exposes triggers for primary alerts, pre-alerts, and all-clear notifications.

Use Homey Flows to turn on lights, notify family, mute media, or run any emergency routine. A Homey widget is included to view live connection state, active alert status, and recent events, with quick enable/disable control for monitoring.
